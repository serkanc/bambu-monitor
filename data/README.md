# Data Directory

This folder stores runtime data and generated artifacts.

## `data/print-cache/`
Temporary files for print transfers and caching.
Safe to clear if corrupted or oversized.

## `data/filament/`
Generated by `tools/filament/*.py`.
Contains filament metadata snapshots used by UI.

## `data/hms/`
Generated by tooling or ingestion steps.
Contains HMS (health monitoring system) data.

Do not hand-edit generated data. Re-run tools instead.
